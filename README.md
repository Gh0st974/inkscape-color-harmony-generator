# Inkscape Color Harmony Generator

A simple and powerful extension for Inkscape to generate color harmonies instantly on your canvas.

![Thumbnail](https://votre-lien-image-si-disponible.png)

## Features
- **Analogous**: 5 colors following the color wheel.
- **Monochrome**: 5 shades of the same hue.
- **Complementary**: Base color and its opposite.
- **Triad**: 3 colors equally spaced on the wheel.
- **Shades (Nuances)**: Dark to light variations of your base color.

## Installation
1. Download the latest release (`.zip` file).
2. Extract the files `color_harmony_generator.py` and `color_harmony_generator.inx`.
3. Copy them into your Inkscape extensions folder:
   - **Linux/macOS**: `~/.config/inkscape/extensions/`
   - **Windows**: `%APPDATA%\inkscape\extensions\`
4. Restart Inkscape.

## How to use
1. Go to **Extensions > Color > Color Harmony Generator**.
2. Pick your base color using the color picker.
3. Choose your harmony type and the size of the squares.
4. Click **Apply**.
